### 2. 源码阅读 ✅

#### UserController

- **接口路径**：`POST /user/user/login`
- **功能**：微信登录，传入 `code`，返回 `id`、`openid`、`token`
- **依赖**：UserService、JwtProperties

#### AddressBookController

| 接口路径 | 方法 | 功能 |
|---|---|---|
| `GET /user/addressBook/list` | list() | 查询当前用户所有地址 |
| `POST /user/addressBook` | save() | 新增地址 |
| `GET /user/addressBook/{id}` | getById() | 根据ID查询地址 |
| `PUT /user/addressBook` | update() | 修改地址 |
| `PUT /user/addressBook/default` | setDefault() | 设置默认地址 |
| `DELETE /user/addressBook` | deleteById() | 删除地址 |
| `GET /user/addressBook/default` | getDefault() | 查询默认地址 |

## 备注

- 项目使用 JWT 实现用户认证，C端拦截器为 `JwtTokenUserInterceptor`
- 地址簿查询时会从 `BaseContext.getCurrentId()` 获取当前登录用户ID
- 地址簿中性别字段：0-女，1-男
- 地址簿中是否默认：0-否，1-是
